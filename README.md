# fetch_and_parse

English
-------
Primary goal:

Download Devconnect events information to feed a favorite AI so it can analyze events by a prompt and user criteria.

Usage (very short):

```bash
pip install playwright
playwright install
python fetch_and_parse.py --url https://luma.com/devconnect --output events_all_extracted.json
```
